
# PrettyMeal- An online mess application

Welcome to PrettyMeal, an innovative Android application designed to revolutionize the way people find and choose their mess meals. With PrettyMeal, customers can explore a wide range of messes, view their menus, and make informed decisions based on their preferences. Mess owners can showcase their offerings and attract more customers. Say goodbye to monotonous meals and embrace a delightful dining experience with PrettyMeal!

## Features

- Secure User Authentication: Register and login with confidence.
- Extensive Mess Listings: Discover a diverse selection of messes in your area.
- Comprehensive Mess Details: Access detailed information about each mess, including menu, location, and contact details.
- Map Integration: Easily locate messes on an interactive map for convenient navigation.
- Membership Options: Choose from Silver and Gold membership plans with varying free meal allocations.
- Personalized Profiles: Update and manage your personal and mess information effortlessly.
- Prompt Support: Access support options for any queries or assistance you may need.

## Installation

1. Clone the repository: `git clone https://github.com/Ajinkya-Kardile/PrettyMeal.git`
2. Create a Google firebase account and import your google-services.json in app directory.  
3. Import the project into Android Studio.
4. Build and run the project on your Android device or emulator.
